25 January, 2015 by Sandra Muller in Blog Posts


OOP

JavaScript is an Object Orientated Programming language. It provides 4 basic capabilities to developers:

  1. Encapsulation - storing relative information together in an object.
  2. Aggregation - storing one object inside of another object.
  3. Inheritance - the capability of a class relying upon another class for some of its properties and methods.
  4. Polymorphism - being able to write a function or method that works in different ways.

Objects are composed of properties. If any of these properties contains a function, it is considered to be a method of the object.

Pretty much everything in JavaScript acts like an object.

If you would like to access the property of the object you can do this in one of three ways:

  • objectName.property ---> tina.firstName

  • objectName["property"] ---> tina["firstName"]

  • objectName[expression] ---> name = "firstName"; tina[name]


    • If you would like to add a property you can use this syntax:

  • objectName.objectProperty = propertyValue ;

  • tina.songTitle = "You're simply the best!" ;


    • If you want to delete a propery you would use this syntax:

  • delete objectName.property ;

  • delete tina.age ;
